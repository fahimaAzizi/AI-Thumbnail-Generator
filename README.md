# 🎨 AI Thumbnail Generator

AI Thumbnail Generator is a web application that uses artificial intelligence to create high-quality, engaging thumbnails from text prompts. Designed for content creators, marketers, and businesses, the platform helps users generate professional thumbnails quickly and efficiently without requiring graphic design experience.

---

## 🚀 Features

* Generate thumbnails using AI-powered image generation
* Create custom thumbnails from text descriptions
* Preview generated thumbnails before downloading
* Fast and responsive user interface
* Mobile-friendly design
* Download generated images in high quality
* Secure API integration

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Thumbnail-Generator.git
cd AI-Thumbnail-Generator
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file and add:

```env
API_KEY=your_api_key
PORT=5000
```

### 4. Start the Development Server

```bash
npm run dev
```

The application will run at:

```text
http://localhost:5000
```

## 📸 How It Works

1. Enter a thumbnail prompt or description.
2. Submit the request to the AI model.
3. The AI generates a thumbnail based on the prompt.
4. Preview the generated image.
5. Download the thumbnail for use on YouTube, social media, blogs, or marketing campaigns.
